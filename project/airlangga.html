Airlangga Ward Profile and Admin Web

This project is part of our internship assignment, which we are working on as a group. We've named it "Airlangga Surabaya Ward Information System," and it encompasses several key features. First, the project provides information about the services offered by Airlangga Ward, making information more accessible to the public. Second, we also share information about activities organized by the village, such as social events and community meetings. The project also includes an employee performance input feature, allowing staff to record their performance and easily generate employee performance reports. With this project, we aim to enhance transparency, efficiency, and information accessibility in Airlangga Ward, while assisting in employee performance management and reporting.
